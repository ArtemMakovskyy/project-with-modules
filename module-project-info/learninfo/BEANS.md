# Курс «Java разработка». Урок 12: Конфигурация



**Аннотации Spring Framework для условной регистрации бинов:**

- `@Conditional` — семейство аннотаций для внедрения бинов на основе условия.
    
- `@ConditionalOnBean` — проверяет наличие определенного бина в контексте.
    
- `@ConditionalOnMissingBean` — проверяет отсутствие определенного бина в контексте.
    
- `@ConditionalOnProperty` — проверяет значение свойства в файле свойств.
